## 深度教学助手前端（Vue 版本）

本目录是一个使用 Vue 3 + Vite 搭建的原型项目，可在聊天界面中演示以下功能：

- 出题示例（生成可编辑题目卡片，模拟保存题库）
- 题库结构概览（展示专题/难度统计）
- 班级/学生管理（查看 mock 班级及学生列表）
- 分配题目（选择班级/学生/专题/难度并模拟调用 `/assign/one`）

所有数据均来自本地 `src/data/mock.js`，方便后续替换为真实 API。

### 本地运行

1. 在 `frontend` 目录下安装依赖：
   ```bash
   npm install
   ```
2. 启动开发服务器：
   ```bash
   npm run dev
   ```
3. 浏览器访问提示地址（默认 `http://localhost:5173`）。

打包发布：
```bash
npm run build
```

构建后的静态文件位于 `dist/`，可部署到任意静态服务器。若需要与 Flask 后端联调，可在 Vite 配置或组件内将 mock 数据替换为真实 REST 接口调用。
