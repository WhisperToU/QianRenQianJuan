# 千人千卷 (AI-Driven Personalized Homework System)

一个 **AI 驱动的个性化作业系统**，为大班额环境下的中学课堂提供 **低成本、可落地** 的因材施教方案。

本项目最初应用于 **雅安市成实外高级中学** 的教学实验，完整设计与初步实践成果见 `report.pdf`。  
目前系统进入第二次实验：面向 **非毕业年级的日常教学**，验证其在真实教学中的持续效果。

---

## ✅ 教学落地价值

市面上已有不少 AI 因材施教产品，但普遍存在以下问题：

- 依赖昂贵硬件  
- 无法与教育局/学校的教学进度匹配  
- 难以真正融入课堂  

**千人千卷** 的核心理念是：

利用 AI 出题能力，由教师掌控“出什么题”“如何分层”“怎样布置”，从而保持教学节奏一致，使 AI 成为教师的放大器而不是替代者。

系统结构非常简单——**一个符合不同年龄教师习惯的前端界面 + 调用大模型 API 出题 + MySQL 存储学校自己的题库**。  
这一套数据库结构中蕴含的 **分层逻辑与教学方法**，可能是整个项目最具创新性的部分。

### 🧩 现实中的两大限制

1. **当前图像识别技术仍不够稳定**：项目非常遗憾没有办法改变批改需要手动的问题；  
2. **题目设计与审核需要学校组织**：只有当学校认识到“建设校本教材”的长期价值时，流程才能顺利运转。好消息是：实际上多数学校管理者已经意识到这一点，因为教育系统宣传了多年，只是不知道怎么投入资源去做这件事情。

为缓解问题 ①，系统采用了以下折中方案：

- 在同一份纸上多次打印：第一次印题、第二次印答案  
- 随机出题时记录每个学生的题目序列，批改界面按顺序显示对应答案，减少教师查找成本  

此外，为适配大班额课堂，项目还配套构建了完整的教学闭环：

**出题 → 批改 → 改错 → 再反馈**

确保流程可落地、教师工作量可计算。

### 📊 初步实验数据（真实课堂）

- 出题效率提升 **70%**  
- 作业适配性提高 **13.1%**  
- 学生深度参与度提升 **44.3%**  
- 对 **中上游学生** 的提升尤为明显
